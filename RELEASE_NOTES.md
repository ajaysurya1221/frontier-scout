# Release Notes: v0.1.0

Frontier Scout v0.1.0 introduces a local-first AI-tool radar and the first
Engineering Scout platform slice.

Highlights:

- CLI radar for AI tools, MCP servers, agent frameworks, and model drops.
- Adoption Firewall commands: `evaluate`, `trial`, `guard`, and `policy init`.
- Incident Change Scout demo with graph-aware retrieval, ReBAC, HITL
  interrupt, trace/audit artifacts, and eval scoring.
- Local deterministic model gateway; no API key required for the demo.
- Security and governance docs mapped to OWASP Agentic Top 10 and NIST AI RMF.
